## 一、线性系统与非线性系统奇点判断、分类

### 1.1 奇点类型判定解题步骤总结

针对平面自治系统：

$$\begin{cases} \frac{dx}{dt} = P(x,y) \\ \frac{dy}{dt} = Q(x,y) \end{cases}$$

**第一步：求奇点 (Singular Points)**

令 $P(x,y)=0$ 且 $Q(x,y)=0$，求解方程组得到所有实数解 $(x_0, y_0)$，即为奇点。

**第二步：线性化与计算雅可比矩阵 (Linearization & Jacobian)**

计算系统在奇点处的雅可比矩阵 $J$：

$$J(x,y) = \begin{pmatrix} P'_x & P'_y \\ Q'_x & Q'_y \end{pmatrix} \quad \Rightarrow \quad A = J(x_0, y_0) = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$$

其中 $a=P'_x(x_0,y_0)$ ，其他同理 。

**第三步：计算特征值 (Eigenvalues)**

求解特征方程 $|\lambda I - A| = 0$，即 $\lambda^2 - p\lambda + q = 0$，其中：

- $p = \text{tr}(A) = a+d = \lambda_1 + \lambda_2$ (迹)
    
- $q = \det(A) = ad-bc = \lambda_1 \lambda_2$ (行列式)
    
    解得特征值 $\lambda_{1,2}$ 。
    

**第四步：奇点分类判别 (Classification)**

根据特征值 $\lambda$ 的性质进行分类（假设 $\det(A) \neq 0$）：

| **分类情形 (Case)** | **特征值情况 (λ)**                                                                 | **具体条件**                    | **奇点类型 (Type)**                                                                              |
| --------------- | ----------------------------------------------------------------------------- | --------------------------- | -------------------------------------------------------------------------------------------- |
| **Case 1**      | **实根，不相等**<br>$\lambda_1, \lambda_2 \in \mathbb{R}, \lambda_1 \neq \lambda_2$ | $\lambda_1 < \lambda_2 < 0$ | **稳定结点** (Stable Node)                                                                       |
|                 |                                                                               | $\lambda_1 > \lambda_2 > 0$ | **不稳定结点** (Unstable Node)                                                                    |
|                 |                                                                               | $\lambda_1 < 0 < \lambda_2$ | **鞍点** (Saddle Point)                                                                        |
| ==**Case 2**==  | ==**实根，相等**<br>$\lambda_1 = \lambda_2 = \lambda$==                            | ==$\lambda < 0$==           | ==**稳定退化结点** (Degenerate Node)== [[微分方程定性方法和数值模拟 (刘正荣编著) (Z-Library).pdf#page=14\|退化结点图 14]] |
|                 | ==_(对应只有一个独立特征向量的情况)_==                                                       | ==$\lambda > 0$==           | ==**不稳定退化结点** (Degenerate Node)==                                                            |
| **Case 3**      | **共轭复根**<br>$\lambda_{1,2} = \alpha \pm i\beta$ ($\alpha \neq 0$)             | $\alpha < 0$                | **稳定焦点** (Stable Focus)                                                                      |
|                 |                                                                               | $\alpha > 0$                | **不稳定焦点** (Unstable Focus)                                                                   |
|                 |                                                                               | $\alpha=0$                  | 线性系统：**中心**（Center）<br>非线性系统：中心待定                                                            |
| ==**Case 4**==  | ==**临界情形** ($b=c=0$)<br>$a=d=\lambda \neq 0$==                                | ==$\lambda < 0$==           | ==**稳定临界结点** (Critical Node)== [[微分方程定性方法和数值模拟 (刘正荣编著) (Z-Library).pdf#page=16\|临界结点点图 16]]  |
|                 | ==_(对应有两个独立特征向量的情况)_==                                                        | ==$\lambda > 0$==           | ==**不稳定临界结点** (Critical Node)==                                                              |
或者按这个表（都一样）：

| **特征值情况 (λ1,2​)**              | **关系 **                 | **奇点类型 (Type)**  | **稳定性 (Stability)**                                 |
| ------------------------------ | ----------------------- | ---------------- | --------------------------------------------------- |
| **实根，异号**                      | $\lambda_1<0<\lambda_2$ | **鞍点 (Saddle)**  | -                                                   |
| **实根，同号**                      | $q > 0, \Delta \ge 0$   | **结点 (Node)**    | $p<0$ 稳定 (Stable)<br><br>$p>0$ 不稳定 (Unstable)       |
| **共轭复根** ($\alpha \pm i\beta$) | $q > 0, \Delta < 0$     | **焦点 (Focus)**   | $p<0$ ($\alpha<0$) 稳定<br><br>$p>0$ ($\alpha>0$) 不稳定 |
| **纯虚根** ($\pm i\beta$)         | $q > 0, p = 0$          | **中心 (Center)**? | **线性系统**：中心<br><br>**非线性系统**：需进一步判断 (见下文)           |

> **注：** $\Delta = p^2 - 4q$。对于退化结点（$\lambda_1=\lambda_2$）和临界结点（只有一个独立特征向量）的情况，笔记中归类为结点

**第五步：检查是否有线性解（Linear Solution）**
若：$\frac{\partial y}{\partial x}=\frac{Q(x,y)}{P(x,y)}=k$
则有 $y=kx$ ,需要画出

**第六步：非线性系统“中心”情形的特殊判断**

当线性化矩阵特征值为纯虚根 ($\lambda = \pm i\beta, \alpha=0$) 时，线性近似判定为中心，但非线性高阶项可能影响稳定性。

- **判断方法**：检查系统是否存在**首次积分 (First Integral)** 或是否为 **哈密顿系统 (Hamiltonian System)**。
    
- **哈密顿系统判据**：若系统满足 $\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} = 0$ (即散度为0)，则为哈密顿系统。
   也就是看是否能通过：$\frac{\partial y}{\partial x}=\frac{Q(x,y)}{P(x,y)}$ 两边积分解出哈密顿方程：$H(x,y)=0$
    
- **结论**：若是哈密顿系统且 $(x_0, y_0)$ 是哈密顿量 $H(x,y)$ 的极值点，则该点为非线性系统的**中心 (Center)**；否则可能是焦点 。
  关于如何判断是否是焦点：[[微分方程定性方法＆数值模拟 笔记.pdf#page=3|手写笔记P3]]

> [!attention] 注意：坐标轴：是$x-y$还是$\xi-\eta$，取决于是否进行了变换

### 1.2 哈密顿系统判别与例题

对于平面自治系统：

$$\begin{cases} \frac{dx}{dt} = P(x,y) \\ \frac{dy}{dt} = Q(x,y) \end{cases}$$

1. 哈密顿系统定义

若存在函数 $H(x,y)$ 使得：
$$\begin{cases}\frac{\partial H}{\partial y}=\frac{dx}{dt} \\
\frac{\partial H}{\partial x}=-\frac{dy}{dt}\end{cases}$$

即满足 $\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} = 0$（散度为0），则称该系统为**哈密顿系统 (Hamiltonian System)** 。
- **性质**：系统的轨线即为哈密顿量 $H(x,y) = C$ 的等高线。
- **中心判别**：若 $(x_0, y_0)$ 是 $H(x,y)$ 的极值点（极大或极小），则该奇点为**中心 (Center)** 。
- **奇点类型：** 绝不可能出现焦点或者节点，只可能出现**中心（center）或者鞍点（saddle）
  >这意味着特征方程通常是 $\lambda^2 + \Delta = 0$ 的形式。特征值 $\lambda$ 只有两种主要情况：
   纯虚数 ($\pm i\omega$)**
   **互为相反数的实数 ($\pm k$)**

**2. 黑板例题解析**
题目：
给定哈密顿函数 $H(x,y)$ 如下，试写出其对应的微分方程组并分析解的性质。
$$H(x,y) = \frac{x^2}{2} + \frac{y^2}{2} - \frac{c x^4}{4} - \frac{a y^4}{4} + \frac{x^2 y^2}{2}$$
**解题步骤：**
第一步：求解微分方程组
根据哈密顿系统定义 $\frac{dx}{dt} = \frac{\partial H}{\partial y}, \frac{dy}{dt} = -\frac{\partial H}{\partial x}$，计算偏导数：
1. 对 $y$ 求偏导：
$$\frac{\partial H}{\partial y} = y - a y^3 + x^2 y = y(1 - a y^2 + x^2)$$
2. 对 $x$ 求偏导：
$$\frac{\partial H}{\partial x} = x - c x^3 + x y^2 = x(1 - c x^2 + y^2)$$

由此得到系统方程：
$$\begin{cases} \frac{dx}{dt} = y(1 - a y^2 + x^2) \\ \frac{dy}{dt} = -x(1 - c x^2 + y^2) \end{cases}$$
第二步：奇点分析......

补充：如何利用全微分法求解（验证）
若题目未直接给出 $H(x,y)$，而是给出方程 $\frac{dy}{dx} = \frac{Q(x,y)}{P(x,y)}$：
$$\frac{dy}{dx} = \frac{-x + cx^3 - xy^2}{y - ay^3 + x^2y}$$
整理为微分形式：
$$(x - cx^3 + xy^2)dx + (y - ay^3 + x^2y)dy = 0$$
检查是否为全微分方程：
$$\frac{\partial}{\partial y}(x + cx^3 + xy^2) = 2xy, \quad \frac{\partial}{\partial x}(y - ay^3 + x^2y) = 2xy$$
两者相等，故存在原函数 $H(x,y)$，积分即可得到上述哈密顿量。

### 2.1 一个零特征根情形

当系统线性化矩阵 $A$ 的特征值出现 **一个为0，另一个不为0**（即 $\det A = 0, \text{tr} A \neq 0$）时，需将其化为标准型利用隐函数定理判断。
#### 2.1.1 化为标准型 (Standard Form)
**原系统**：
$$\begin{cases} x' = ax + by + \varphi(x,y) \\ y' = cx + dy + \psi(x,y) \end{cases}$$
其中特征方程 $\lambda^2 - (a+d)\lambda + (ad-bc) = 0$ 有一零根，即 $ad-bc=0, a+d \neq 0$。
变换方法（依据 Proposition 3.3）：
根据系数情况选择以下变换，将系统化为标准型：
$$\begin{cases} \frac{d\xi}{d\tau} = p_2(\xi, \eta) \\ \frac{d\eta}{d\tau} = \eta + q_2(\xi, \eta) \end{cases}$$
_(注：$p_2, q_2$ 为高阶项，无线性项)_

| **原始系数条件**                      | **变换公式**                                  | **时间变换**        |
| ------------------------------- | ----------------------------------------- | --------------- |
| **(i)** $b \neq 0$              | $\xi = -dx + by, \quad \eta = ax + by$    | $\tau = (a+d)t$ |
| **(ii)** $b = 0, d \neq 0$      | $\xi = x, \quad \eta = \frac{c}{d}x + y$  | $\tau = dt$     |
| **(iii)** $b=0, d=0 (a \neq 0)$ | $\xi = -\frac{c}{a}x + y, \quad \eta = x$ | $\tau = at$     |

#### 2.1.2 奇点性质判别 (Proposition 3.5)

对于标准型系统（将 $\xi, \eta$ 记为 $x, y$）：

$$\begin{cases} x' = p_2(x, y) \\ y' = y + q_2(x, y) \end{cases}$$
**步骤：**
1. ==由第二个方程 $y + q_2(x,y) = 0$ 利用隐函数定理求出 **$y = \varphi(x)$==** （满足 $\varphi(0)=\varphi'(0)=0$）。
2. 将 $y = \varphi(x)$ 代入第一个方程，得到：$$p_2(x, \varphi(x)) = a_m x^m + a_{m+1}x^{m+1} + \dots \quad (a_m \neq 0)$$
3. **判别法则**

| **m 的奇偶性**  | **$a_m$​ 的符号** | **奇点类型 (Figure 3.23, 3.24)** | **备注**                    |
| ----------- | -------------- | ---------------------------- | ------------------------- |
| **$m$ 为奇数** | $a_m > 0$      | **不稳定结点 (Unstable Node)**    |                           |
| **$m$ 为奇数** | $a_m < 0$      | **鞍点 (Saddle)**              |                           |
| **$m$ 为偶数** | $a_m > 0$      | **鞍结点 (Saddle-node)**        | 抛物型区域在**右侧** (Right side) |
| **$m$ 为偶数** | $a_m < 0$      | **鞍结点 (Saddle-node)**        | 抛物型区域在**左侧** (Left side)  |

### 2.2 两个零特征根的情形 (System with Two Zero Eigenvalues)

当系统线性化矩阵 $A$ 的特征值 **全为0**（即 $\det A = 0, \text{tr} A = 0$）且非零矩阵时，对应 Jordan 块形式。
#### 2.2.1 化为标准型 (Standard Form)
变换方法（依据 Proposition 3.4）：
将原系统变换为标准型：
$$\begin{cases} \xi' = \eta + p_2(\xi, \eta) \\ \eta' = q_2(\xi, \eta) \end{cases}$$

| ---------------------------------- | --------------------------------- |
| **(i)** $a \neq 0$                 | $\xi = -y, \quad \eta = -cx + ay$ |
| **(ii)** $a = 0, b \neq 0$         | $\tau = bt$ (时间变换)                |
| **(iii)** $a = 0, b = 0, c \neq 0$ | $\xi = y, \quad \eta = cx$        |

#### 2.2.2 奇点性质判别 (Proposition 3.6 & Tables 3.1-3.2)

$$\begin{cases} x' = y + p_2(x, y) \\ y' = q_2(x, y) \end{cases}$$

## 三、高阶系统奇点判别（只考$G(\theta)=0,y=ux$）
### 3.1 高阶系统定义与适用情形 
针对系统：
$$\begin{cases} \frac{dx}{dt} = X_m(x,y) + \varphi(x,y) \\ \frac{dy}{dt} = Y_m(x,y) + \psi(x,y) \end{cases}$$
其中 $X_m, Y_m$ 是 $m$ 次齐次多项式（$\bbox[yellow]{m≥2}$），$\varphi, \psi$ 是更高阶的项
$$G(\theta) \equiv 0 \quad \Leftrightarrow \quad x Y_m(x,y) - y X_m(x,y) \equiv 0$$
即最低次项部分满足比例关系时，使用此方法 。

> [!example]- 区分非线性系统和高阶系统的例子
> 
> $\begin{cases} x' = xy \\ y' = y - x^2 \end{cases}$ （非线性系统）
> 
> $\begin{cases} x' = xy \\ y' = y^2 - x^4 \end{cases}$（高阶系统）
>

$$\bbox[yellow]{ G(\theta) \equiv \cos\theta \cdot Y_m(\cos\theta, \sin\theta) - \sin\theta \cdot X_m(\cos\theta, \sin\theta)}$$
> [!tip]- ##### 什么是 $G(\theta)$？
> $G(\theta)$ 是在处理高阶奇点时，通过引入**极坐标变换** ($x = r\cos\theta, y = r\sin\theta$) 导出的一个核心函数，用于判断轨线的切线方向（Characteristic Directions）。
> 推导来源：
> 考虑高阶系统的主要部分 $X_m, Y_m$，在极坐标下计算幅角 $\theta$ 的变化率：
> $$ \frac{d\theta}{dt} = \frac{x \frac{dy}{dt} - y \frac{dx}{dt}}{r^2} \approx r^{m-1} G(\theta)$$
> 由此定义：
> $$\bbox[yellow]{ G(\theta) \equiv \cos\theta \cdot Y_m(\cos\theta, \sin\theta) - \sin\theta \cdot X_m(\cos\theta, \sin\theta)}$$
> （直观理解：$G(\theta)$ 来源于 $xY_m - yX_m$ 的极坐标表达）。
### 3.2 解题步骤

1. 引入变换：
    令 $y = ux$，则导数关系为：$$\frac{dy}{dx} = u + x\frac{du}{dx}$$
2. 代入方程：
    利用 $\frac{dy}{dx} = \frac{Y_m + \psi}{X_m + \varphi}$，将 $y=ux$ 代入右端：$$u + x\frac{du}{dx} = \frac{Y_m(x,ux) + \psi(x,ux)}{X_m(x,ux) + \varphi(x,ux)}$$
3. **化简求解**：
    
    - 利用齐次性质提取 $x^m$ 并化简。
        
    - 由于 $xY_m - yX_m \equiv 0$，方程通常可化简为关于 $x$ 和 $u$ 的可分离变量方程或低阶系统。
        
    - 得到形式如 $\frac{du}{dx} = f(x,u)$ 的方程。
        
4. 分析 $x-u$ 相平面：
    
    在 $x-u$ 平面上求解微分方程的轨线或判断奇点类型（如中心、结点等）。
    
5. 还原相图：
    
    根据 $y=ux$ 的几何意义，将 $x-u$ 平面的相图映射回 $x-y$ 平面，确定原系统奇点的性质（注意 $u = y/x$ 表示斜率）。

## 四、极限环与分叉

### 4.1 极限环的判定与求解 (极坐标变换法)

针对形如 $\begin{cases} \dot{x} = -y + x f(r) \\ \dot{y} = x + y f(r) \end{cases}$ 或包含 $x^2+y^2$ 结构的系统，使用极坐标变换是最有效的方法。
#### 4.1.1 解题通用步骤

第一步：引入极坐标变换
令 $\begin{cases} x = r\cos\theta \\ y = r\sin\theta \end{cases}$，利用公式：

$$r\frac{dr}{dt} = x\frac{dx}{dt} + y\frac{dy}{dt} \quad \Rightarrow \quad \bbox[yellow]{\frac{dr}{dt} = \frac{x\dot{x} + y\dot{y}}{r}}$$

$$\bbox[yellow]{\frac{d\theta}{dt} = \frac{x\dot{y} - y\dot{x}}{r^2}}$$
>[!quote]- 这个公式怎么来的？
 **推导过程**
 利用直角坐标 $(x,y)$ 与极坐标 $(r,\theta)$ 的关系：
$$r^2 = x^2 + y^2, \quad \tan\theta = \frac{y}{x}$$
 对 $r^2 = x^2 + y^2$ 两边关于时间 $t$ 求导：$$2r \cdot \frac{dr}{dt} = 2x \cdot \frac{dx}{dt} + 2y \cdot \frac{dy}{dt}$$$$\Rightarrow \dot{r} = x \dot{x} + y \dot{y} \quad \Rightarrow \quad \boxed{\frac{dr}{dt} = \frac{x\dot{x} + y\dot{y}}{r}}$$
 对 $\tan\theta = \frac{y}{x}$ 两边关于时间 $t$ 求导：$$\sec^2\theta \cdot \frac{d\theta}{dt} = \left( \frac{y}{x} \right)' = \frac{\dot{y}x - y\dot{x}}{x^2}$$$$\Rightarrow\frac{r^2}{x^2} \cdot \frac{d\theta}{dt} = \frac{x\dot{y} - y\dot{x}}{x^2}$$
 两边同时乘以 $x^2$ 并除以 $r^2$：
$$\boxed{\frac{d\theta}{dt} = \frac{x\dot{y} - y\dot{x}}{r^2}}$$


第二步：代入消元化简
将原方程代入上述公式，利用 $x^2+y^2=r^2$ 化简，目标是得到变量分离的方程，通常形式为：
$$\begin{cases} \frac{dr}{dt} = r \cdot G(r) \\ \frac{d\theta}{dt} = H(r,\theta) \quad (\text{通常常数，如 } 1) \end{cases}$$

第三步：求解极限环半径
令 $\frac{dr}{dt} = 0$ (且 $r \neq 0$)，解出的实根 $r = r_0$ 即为极限环的半径。

- 若无正实根，则不存在极限环。
    
- 若有多个正实根，则存在多个极限环。
    

第四步：判断稳定性 (Stability)
画出 $\frac{dr}{dt}$ 关于 $r$ 的相图（即一维 $r$ 轴上的箭头图），检查在 $r_0$ 附近的符号变化：

- **稳定极限环 (Stable)**: 当 $r < r_0$ 时 $\frac{dr}{dt} > 0$ (扩张)，当 $r > r_0$ 时 $\frac{dr}{dt} < 0$ (收缩)。即 $r$ 趋向于 $r_0$。
    
- **不稳定极限环 (Unstable)**: 当 $r < r_0$ 时 $\frac{dr}{dt} < 0$，当 $r > r_0$ 时 $\frac{dr}{dt} > 0$。即 $r$ 远离 $r_0$。
    
- **半稳定极限环 (Semi-stable)**: 两侧符号相同。

#### 4.1.2 典例解析 (依据手写笔记)

##### 例 1：稳定极限环
系统：
$$\begin{cases} \frac{dx}{dt} = -y + x(1 - x^2 - y^2) \\ \frac{dy}{dt} = x + y(1 - x^2 - y^2) \end{cases}$$

**解析**：
1. 变换：$$x\dot{x} + y\dot{y} = x[-y + x(1-r^2)] + y[x + y(1-r^2)] = (x^2+y^2)(1-r^2) = r^2(1-r^2)$$$$\Rightarrow \quad r\frac{dr}{dt} = r^2(1-r^2) \quad \Rightarrow \quad \frac{dr}{dt} = r(1-r^2)$$
    同理求得 $\frac{d\theta}{dt} = 1$。
2. 求极限环：
    令 $\frac{dr}{dt} = 0 \Rightarrow r(1-r)(1+r) = 0$。
    因为 $r>0$，解得唯一正根 $r=1$。即存在一个单位圆形状的极限环。
3. **判断稳定性**：
    - 当 $0 < r < 1$ 时（环内）：$1-r^2 > 0 \Rightarrow \dot{r} > 0$ (轨迹向外螺旋)。
        
    - 当 $r > 1$ 时（环外）：$1-r^2 < 0 \Rightarrow \dot{r} < 0$ (轨迹向内螺旋)。
        
    - **结论**：$r=1$ 是**稳定极限环 (Stable Limit Cycle)**。
##### 例 2：不稳定极限环
系统：
$$\begin{cases} \frac{dx}{dt} = -y + x(x^2 + y^2 - 1) \\ \frac{dy}{dt} = x + y(x^2 + y^2 - 1) \end{cases}$$
**解析**：
1. **变换**：得到 $\frac{dr}{dt} = r(r^2 - 1)$。
    
2. **稳定性**：
    
    - $r < 1 \Rightarrow \dot{r} < 0$ (向原点收缩)。
        
    - $r > 1 \Rightarrow \dot{r} > 0$ (发散)。
        
    - **结论**：$r=1$ 是**不稳定极限环 (Unstable Limit Cycle)**。此时原点是稳定的。

### 4.2 分岔
#### 4.2.1 Hopf 分岔 (Hopf Bifurcation)

==**一句话理解**：**“无中生有”** —— 一个**平衡点**失去稳定性，变成了一个**极限环**。==
 **核心定义 (考试判据)：**
- **发生位置**：在**奇点（平衡点）**处。
- **触发条件（$\mu$ 为系统中的参数）**：
    1. 线性化矩阵有一对**复特征值** $\lambda = \alpha(\mu) \pm i\beta(\mu)$。
        
    2. 随着参数 $\mu$ 变化，实部 $\alpha(\mu)$ **穿过虚轴**（即 $\alpha=0$）（形成中心（极限环））。
        
    3. 穿过时速度不为0（横截性条件）。
        
- **结果**：奇点由稳定变为不稳定（或反之），并 **“吐出”一个极限环**。
>[!tip]- 完整定义：
对于一个含参系统：$$\begin{cases} \frac{dx}{dt} = P(x,y,\alpha) \\ \frac{dy}{dt} =Q(x,y,\alpha) \end{cases}$$
若$(x_0,y_0)$为孤立奇点，且存在$\alpha_0$: $\alpha>\alpha_0$和$\alpha＜\alpha_0$时稳定性相反，且诞生了一个极限环，则为hopf分岔。
#### 4.2.2 多重分岔 (Multiple Bifurcation )
==**一句话理解**：**“一分为二”** —— 一个**半稳定极限环**分裂成了**两个**普通极限环，或者消失了。==
**核心定义 (考试判据)**
- **发生位置**：在**已有的周期轨道（闭轨）附近，而不是在奇点处。**
- 触发条件：
    庞加莱映射（Poincaré Map）的导数性质改变。
    
    - **重数 $k=2$**：在临界时刻，位移函数 $d'(s) = 0$ 且 $d''(s) \neq 0$。
        
    - 这代表曲线与 $x$ 轴**相切**（二重根）。
        
- **结果**：
    
    - 参数稍变，相切变成**两个交点** $\rightarrow$ 两个极限环。
        
    - 参数反向变，相切变成**无交点** $\rightarrow$ 极限环消失。
 
|**维度**|**Hopf 分岔**|**多重分岔 (重极限环)**|
|---|---|---|
|**主角**|**奇点** (点)|**极限环** (圈)|
|**过程**|点 $\to$ 点 + 圈|1个圈 $\to$ 2个圈 (或消失)|
|**判定数学具**|**特征值** (Eigenvalues)|**庞加莱映射导数** ($d'(s)$)|
|**判定关键**|实部 $\alpha = 0$|导数 $d'(s) = 0$|
|**直观感觉**|这种分岔是**“诞生”**|这种分岔是**“分裂”**|

#### 4.2.3 分岔典例解析
[[微分方程定性方法＆数值模拟 笔记.pdf#page=15|笔记P15]]
考察方程：

$$\begin{cases} \frac{dx}{dt} = -y - \epsilon x(x^2 + y^2 - 1 - \epsilon) \\ \frac{dy}{dt} = x - \epsilon y(x^2 + y^2 - 1 - \epsilon) \end{cases}$$
其中 $\epsilon$ 是分岔参数。
**步骤解析**：
1. 极坐标变换：
$$x\dot{x} + y\dot{y} = -\epsilon (x^2+y^2)(r^2 - 1 - \epsilon) = -\epsilon r^2 (r^2 - 1 - \epsilon)$$$$\Rightarrow \quad \frac{dr}{dt} = -\epsilon r [r^2 - (1 + \epsilon)]$$
$$\frac{d\theta}{dt} = 1$$
2. 寻找平衡点与极限环 ($r \ge 0$)：
    令 $\dot{r} = 0$，解得：
    - $r = 0$ (原点，始终存在)
        
    - $r^2 = 1 + \epsilon$。此解存在的条件是 $1 + \epsilon > 0$，即 $\epsilon > -1$。
        
3. **分情况讨论参数 $\epsilon$**：
    - 情形 1：$\epsilon \le -1$
        
        方程 $r^2 = 1+\epsilon$ 无实根（或为0）。
        
        $\dot{r} = -\epsilon r^3 + \dots > 0$。
        
        结论：无极限环，原点不稳定，所有轨迹发散。
        
    - 情形 2：$-1 < \epsilon < 0$
        
        存在极限环 $r = \sqrt{1+\epsilon}$。
        
        分析符号：$-\epsilon > 0$。
        
        - 当 $r < \sqrt{1+\epsilon}$ 时，$[r^2 - (1+\epsilon)] < 0 \Rightarrow \dot{r} < 0$。
            
        - 注意：这里笔记中的正负号需仔细检查。若 $\epsilon < 0$，则 $-\epsilon > 0$。
            
            若 $r$ 很小，$\dot{r} \approx -\epsilon r (-(1+\epsilon)) > 0$？
            
            修正依据笔记推导：
            
            令 $f(r) = -\epsilon r [r^2 - (1+\epsilon)]$。若 $\epsilon \in (-1, 0)$：
            
        - $r < \sqrt{1+\epsilon}$ (环内): $\dot{r} > 0$ (离开发散)。
            
        - $r > \sqrt{1+\epsilon}$ (环外): $\dot{r} < 0$ (向内收缩)。
            
            结论：产生一个稳定极限环 $r = \sqrt{1+\epsilon}$。原点变得不稳定。
            
    - 情形 3：$\epsilon > 0$
        
        存在极限环 $r = \sqrt{1+\epsilon}$。
        
        分析符号：$-\epsilon < 0$。
        
        - $r < \sqrt{1+\epsilon}$ (环内): $[r^2 - (1+\epsilon)] < 0 \Rightarrow \dot{r} > 0$。
            
        - 结论：产生一个稳定极限环。

## 五、无穷远奇点（I型和II型）
### 5.1 基本概念
在平面自治系统中，为了了解轨道延伸到无穷远处的走向（例如是否存在包含无穷远点的闭轨，或者轨道是否从无穷远出发），我们需要将 $x-y$ 平面映射到**庞加莱球面 (Poincaré Sphere)** 上。

- **无穷远点**: 对应庞加莱球面的**赤道**。
    
- **分析方法**: 由于直接在球面上分析不便，通常利用**中心投影法 (Central Projection)**，将球面分别投影到两个切平面上，从而引出两种变换（I型和II型）。
![[Pasted image 20260113105929.png]]
### 5.2 两个核心变换 (必考公式)

针对系统：
$$\begin{cases} \frac{dx}{dt} = P(x,y) \\ \frac{dy}{dt} = Q(x,y) \end{cases}$$

假设 $P, Q$ 的最高次数为 $m$。
#### 1. 第一型变换 (Transformation I): $x = 1/z, \; y = u/z$
- **几何意义**: 将球面投影到 $x=1$ 的切平面上。
    
- **适用范围**: ==能够分析除了 $y$ 轴方向以外的所有无穷远点==。主要用于寻找和分析分布在 $x$ 轴方向及斜率有限方向上的无穷远奇点。

- 变换公式:$$\bbox[yellow]{x = \frac{1}{z}, \quad y = \frac{u}{z}}$$
    其中 $z=0$ 代表无穷远。
- 推导后的系统:
    将变换代入原方程，并作时间变换 $d\tau = \frac{dt}{z^{m-1}}$ (去分母，分母为 $z^{m-1}$，也就是原方程最高次减1)，得到关于 $u$ 和 $z$ 的方程组：
$$\begin{cases} \frac{du}{d\tau} =Q(1, u) - u P(1, u)= \hat P(u, z) \\ \frac{dz}{d\tau} = -z P(1, u)= \hat Q(u, z) \end{cases} $$
    (注：公式中 $P(1, u)$ 指将原多项式中的 $x$ 换成 1，$y$ 换成 $u$ 后的最高次项部分，具体视去分母后的结果而定)

#### 2. 第二型变换 (Transformation II): $x = v/z, \; y = 1/z$

- **几何意义**: 将球面投影到 $y=1$ 的切平面上。
    
- **适用范围**: ==**专门用于分析 $y$ 轴方向**的无穷远奇点（这是第一型变换的盲区）==。
    
- 变换公式:
 $$\bbox[yellow]{x = \frac{v}{z}, \quad y = \frac{1}{z}}$$
- 推导后的系统:
    
    同样代入并作时间变换 $d\tau = \frac{dt}{z^{m-1}}$，得到关于 $v$ 和 $z$ 的方程组：$$\begin{cases} \frac{dv}{d\tau} = P(v, 1) - v Q(v, 1)=\hat P(v, z) \\ \frac{dz}{d\tau} = -z Q(v, 1) =\hat Q(v, z)\end{cases}$$
### 5.3 解题步骤与技巧

**第一步：变换与降次 (Transformations)

对原系统 $\begin{cases} \dot{x} = P(x,y) \\ \dot{y} = Q(x,y) \end{cases}$ 应用两种变换，并对时间变换然后去分母。

**第二步：局部奇点分析
在 $u-z$ 平面（或 $v-z$ 平面）上分析：
1.  **求奇点位置**:
     ==I型（$u-z$）只看 $z=0$ 的奇点，也就是 $u$ 轴上的奇点==，若有则存在除了y轴以外上的无穷远奇点。
     ==II型（$v-z$）只看$(0,0)$处的奇点==，若有则存在y轴上的无穷远奇点。
2.  **判断奇点类型**:
    * 计算雅可比矩阵 $J$ 在 $(u_i, 0)$ 处的值。
    * 利用特征值 $\lambda_1, \lambda_2$ 判定类型。
    * **画出奇点草图**

**第三步：综合绘图与映射 (Mapping & Global Portrait)

将$u-z$ 和 $v-z$ 的奇点草图进行时间逆变换后，在对每个象限进行分区

**⚠️ 常见陷阱速查 (Checklist)
- [ ] **时间变换是否做对？** (二次系统除以 $z$，三次系统除以 $z^2$)
- [ ] **是否遗漏了 y 轴方向？** (必须检查 II 型变换的 $v=0$)
- [ ] **I型只需考虑u轴上的奇点，II型只需考虑原点是不是奇点**
- [ ] **有没有对时间进行逆转换（$t=\tau z^{m-1}$）
- [ ] **$m$ 的奇偶性看准了吗？** (决定了对径点是同向还是反向)
- [ ] **扇形区方向对吗？** (如果是鞍结节点，注意抛物型扇形区的位置)


> [!important] 注意时间方向的变化
> 
> 在作时间变换 $d\tau = \frac{dt}{z^{m-1}}$ 时：
> 
> - 若系统最高次数 $m$ 为**偶数**（如 $m=2$）：$m-1$ 为奇数。当 $z$ 变号时（从上半球面到下半球面），时间方向会反转。
>     
> - 若系统最高次数 $m$ 为**奇数**（如 $m=3$）：$m-1$ 为偶数。$z^{m-1} > 0$，时间方向保持不变。
>     
> - **结论**: 奇点性质（如稳定/不稳定）在赤道对面的点上是否相反，取决于 $m$ 的奇偶性。

## 六、用定性方法求解偏微分方程

[[微分方程定性方法＆数值模拟 笔记.pdf#page=19|笔记P19]]
### 6.1 什么是 KdV 方程

KdV（Korteweg–de Vries）方程是描述**弱非线性 + 弱色散**长波传播的经典模型。常见标准形：  
$$u_t+6uu_x+u_{xxx}=0.$$  
更一般（与你笔记一致的写法）：  
$$\boxed{u_t+buu_x+u_{xxx}=0},\qquad b\neq 0.$$  
其中：
- $u(x,t)$：波形（如水面高度、等离子体密度扰动等）
    
- $buu_x$：非线性对流项（陡峭化）
    
- $u_{xxx}$：色散项（展开波形）
    
### 6.2 什么是行波变换
行波假设：波形以速度 $c$ 不变形传播：  
$$u(x,t)=\varphi(\xi),\qquad \xi=x-ct$$  
则导数变换为  
$$u_t=-c\varphi',\quad u_x=\varphi',\quad u_{xxx}=\varphi''' \qquad('=\frac{d}{d\xi}).$$

代回一般 KdV：  
$$-c\varphi' + b\varphi\varphi' + \varphi'''=0$$  
整理为  
$$\varphi'''+(b\varphi-c)\varphi'=0.$$


### 6.3.首次积分：把三阶 ODE 降到二阶，再变成相平面系统

对上式对 $\xi$ 积分一次：  
$$\boxed{\varphi''-c\varphi+\frac b2\varphi^2=A.}$$  
令  
$$\bbox[yellow]{y=\varphi'}$$  
得到平面自治系统（你笔记写“$\varphi-y$ 相图”）：  
$$  
\boxed{  
\begin{cases}  
\varphi' = y,\\  
y' = c\varphi-\dfrac b2\varphi^2 + A.  
\end{cases}}  
$$

> 你笔记中的 $y' = c\varphi-3\varphi^2$ 是取 $b=6$、$A=0$ 的特例。

---

### 6.4 Hamiltonian 首次积分（能量守恒形式）

把二阶方程乘以 $\varphi'$ 并积分（或直接对系统找积分因子），可得守恒量：  
$$  
\boxed{  
H(\varphi,y)  
=\frac12y^2-\frac c2\varphi^2+\frac b6\varphi^3-A\varphi  
=h.}  
$$  
等价地写为（更适合画相图）：  
$$  
\boxed{  
y^2  
= c\varphi^2-\frac b3\varphi^3+2A\varphi+2h  
=:F(\varphi).}  
$$  
相图轨线就是 $H(\varphi,y)=h$ 的等能曲线；允许运动区间由 $y^2\ge 0\iff F(\varphi)\ge 0$ 给出。



### 6.5 平衡点与类型

平衡点满足 $y=0,;y'=0$，即  
$$c\varphi-\frac b2\varphi^2+A=0.$$  
**在你笔记主要讨论的 $A=0$ 情况：**  
$$(\varphi,y)=(0,0),\quad \left(\frac{2c}{b},0\right).$$

Jacobian：  
$$  
J=  
\begin{pmatrix}  
0&1\  
c-b\varphi&0  
\end{pmatrix}.  
$$

- 在 $(0,0)$：特征值 $\lambda=\pm\sqrt{c}$（当 $c>0$）$\Rightarrow$ **鞍点(Saddle)**
    
- 在 $(2c/b,0)$：此处 $c-b\varphi=c-2c=-c$，特征值 $\lambda=\pm i\sqrt{c}$（当 $c>0$）$\Rightarrow$ **中心(Center)**
 
**轨线对应:**
  - **同宿轨 (Homoclinic Orbit):** 从鞍点出发回到鞍点的封闭曲线 $\longleftrightarrow$ **孤立波解**。
  - **周期轨 (Periodic Orbit):** 围绕中心的闭合曲线 $\longleftrightarrow$ **周期波解**。


### 6.6 用“势能”看能量阈值（5种势能情形）

把 Hamiltonian 写成  
$$H=\frac12y^2+V(\varphi),\quad  
V(\varphi)= -\frac c2\varphi^2+\frac b6\varphi^3-A\varphi.$$  
在 $A=0$ 下，$V(\varphi)$ 在 $\varphi=\frac{2c}{b}$ 处取极小值：  
$$h_{\min}=V\Big(\frac{2c}{b}\Big)= -\frac{2c^3}{3b^2}.$$ 
若取 $b=6$，就得到你笔记的  
$$h_{\min}=-\frac{c^3}{54}.$$

于是能量水平 $h$ 的典型分类（对应你表格的 5 行）：

1. **$h<h_{\min}$**：无实轨线（$y^2<0$），无对应实行波
    
2. **$h=h_{\min}$**：退化为中心平衡点（常值解）
    
3. **$h_{\min}<h<0$**：中心附近闭轨 $\Rightarrow$ 周期行波（椭圆函数/ cnoidal wave）
    
4. **$h=0$**：通过鞍点的同宿轨 $\Rightarrow$ 孤立子（$\mathrm{sech}^2$）；另一个分支给出奇异解（$\mathrm{csch}^2$）
    
5. **$h>0$**：开轨（无界/单调型），对应波形不有界或含奇异（通常物理上不取）

### 6.7  轨线 $\Gamma_i$ 与原 PDE 解的对应

#### 6.7.1 $\Gamma_1$：同宿轨（孤立子 / solitary wave，对应 $h=0$，$\varphi\ge 0$ 分支）


当 $h=0$ 时，  
$$  
y^2=c\varphi^2-\frac b3\varphi^3  
=\frac b3\varphi^2\Big(\frac{3c}{b}-\varphi\Big).  
$$  
取 $\varphi\ge 0$ 且 $0\le \varphi\le \frac{3c}{b}$ 的分支：  
$$  
\frac{d\varphi}{d\xi}  
=\pm \sqrt{\frac b3}\varphi\sqrt{\frac{3c}{b}-\varphi}.  
$$  
积分（标准分离变量）可得经典孤立子：  
$$  
\boxed{  
\varphi(\xi)=\frac{3c}{b}\mathrm{sech}^2\Big(\frac{\sqrt c}{2},\xi\Big)}.  
$$  
回到原变量 $\xi=x-ct$：  
$$  
\boxed{  
u(x,t)=\frac{3c}{b}\mathrm{sech}^2\Big(\frac{\sqrt c}{2}(x-ct-\xi_0)\Big)}.  
$$

> 取 $b=6$ 就是你笔记里的 $u=\dfrac c2 \mathrm{sech}^2(\frac{\sqrt c}{2}(x-ct))$。


#### 6.7.2 $\Gamma_2$：同一能量的另一支（奇异波 / blow-up 型，对应 $h=0$，$\varphi\le 0$ 分支）

仍然 $h=0$，但走到 $\varphi<0$ 的允许区，会出现分母为 0 的积分结构，对应你笔记写出的：  
$$  
\boxed{  
\varphi(\xi)=-\frac{3c}{b}\mathrm{csch}^2\Big(\frac{\sqrt c}{2}\xi\Big)}  
$$  
即  
$$  
\boxed{  
u(x,t)=-\frac{3c}{b}\mathrm{csch}^2\Big(\frac{\sqrt c}{2}(x-ct)\Big)}.  
$$  
它在 $\xi=0$ 有不可去奇异，一般当作数学分支存在（物理上通常不取）。

#### 6.7.3 $\Gamma_3$：中心附近闭轨（周期波 / cnoidal wave，对应 $h_{\min}<h<0$，$\varphi\ge 0$）

当 $h_{\min}<h<0$ 时，三次多项式 $F(\varphi)$ 有三个实根。按你笔记写法记为  
$$\alpha>\beta>\gamma,$$  
并把  
$$  
F(\varphi)=\frac b3(\alpha-\varphi)(\varphi-\beta)(\varphi-\gamma)  
$$  
（这样最高次项系数是 $-\frac b3\varphi^3$，与上式匹配）。

于是  
$$  
\frac{d\varphi}{d\xi}  
=\pm\sqrt{\frac b3}\sqrt{(\alpha-\varphi)(\varphi-\beta)(\varphi-\gamma)}.  
$$

对闭轨（$\varphi$ 在 $[\beta,\alpha]$ 往返振荡），标准积分给出（与你笔记一致的 $\mathrm{sn}^2$ 形式）：  
$$  
\boxed{  
\varphi(\xi)=\alpha-(\alpha-\beta),  
\mathrm{sn}^2\Big(\kappa\xi,m_1\Big)},  
$$  
其中  
$$  
\boxed{  
m_1=\frac{\alpha-\beta}{\alpha-\gamma}},\qquad  
\boxed{  
\kappa=\sqrt{\frac{b(\alpha-\gamma)}{12}} }.  
$$  
回到原变量：$\xi=x-ct$。

> 取 $b=6$ 时，$\kappa=\sqrt{\frac{\alpha-\gamma}{2}}$，就为笔记里的系数形式。

#### 6.7.4 $\Gamma_4$：另一族（仍来自 $h_{\min}<h<0$ 的能级，$\varphi\le 0$）

在同一能级上，除了围绕中心的闭轨，还会有落在另一允许区间的轨线（你笔记给出的是一种等价的 Jacobi 形式，写成分式）：

$$  
\boxed{  
\varphi(\xi)=  
\frac{\gamma-\beta\mathrm{sn}^2(\kappa\xi,m_1)}  
{1-\mathrm{sn}^2(\kappa\xi,m_1)}  
}  
$$  
（参数 $m,\kappa$ 仍由 $\alpha,\beta,\gamma$ 决定。和6.7.3一样）

这类解往往对应**非对称周期型**或带“尖/陡”结构的周期行波分支（从相图上看是另一块允许区域上的运动）。


#### 6.7.5 $\Gamma_5$：$h=h_{\min}$（退化闭轨 $\Rightarrow$ 常值解，$\varphi ＜  0$）

当 $h=h_{\min}$ 时，轨线收缩为中心点：  
$$  
\varphi(\xi)= \frac{2c}{b}  
\quad\Rightarrow\quad  
\boxed{u(x,t)\equiv \frac{2c}{b}}.  
$$  
取 $b=6$ 就是 $u\equiv c/3$。

#### 6.7.6 $\Gamma_6$：$h<h_{\min}$（无实行波）

此时 $F(\varphi)<0$ 导致 $y^2<0$，相图无实轨线，对应无实值行波解（与你笔记最上行一致）。


#### 6.7.7 $\Gamma_7$：$h>0$（开轨：无界/奇异/单调型）

相图上出现开轨，$\varphi(\xi)$ 一般会走向无界或在有限 $\xi$ 发生奇异。你笔记表格里这一行没写出封闭公式是合理的：
- 这类分支常不作为“物理可接受的有界波形”，但定性上要知道它对应开轨。

### 6.8 完整解题步骤模板”（适用于一般 KdV）

**Step 1：写出一般 KdV

$$u_t+buu_x+u_{xxx}=0.$$

**Step 2：行波变换

$$u(x,t)=\varphi(\xi),\quad \xi=x-ct.$$  
得到三阶 ODE：  
$$-c\varphi'+b\varphi\varphi'+\varphi'''=0.$$

**Step 3：首次积分（A一般为0？）

$$\varphi''-c\varphi+\frac b2\varphi^2=A.$$

**Step 4：化为相平面系统

$$\varphi'=y,\qquad y'=c\varphi-\frac b2\varphi^2+A.$$

**Step 5：写 Hamiltonian，并给出轨线方程

$$H(\varphi,y)=\frac12y^2-\frac c2\varphi^2+\frac b6\varphi^3-A\varphi=h,$$  
$$y^2=F(\varphi)=c\varphi^2-\frac b3\varphi^3+2A\varphi+2h.$$

**Step 6：相图定性（关键是根与能量阈值）

- 平衡点：解 $c\varphi-\frac b2\varphi^2+A=0$
    
- 线性化判断鞍/中心/结点
    
- 用 $F(\varphi)\ge 0$ 找允许区间
    
- 随 $h$ 变化画出（或描述）$\Gamma_i$ 的拓扑结构：闭轨/同宿/开轨
    

**Step 7：把每类轨线“反推成解”
沿轨线用  
$$\frac{d\varphi}{d\xi}=\pm\sqrt{F(\varphi)}$$  
做分离变量积分：
- 同宿轨 $\Rightarrow$ $\mathrm{sech}^2$ 孤立子（$\Gamma_1$）
    
- 另一同能分支 $\Rightarrow$ $\mathrm{csch}^2$ 奇异解（$\Gamma_2$）
    
- 三实根闭轨 $\Rightarrow$ Jacobi 椭圆函数周期波（$\Gamma_3,\Gamma_4$）
    
- 平衡点 $\Rightarrow$ 常值解（$\Gamma_5$）
    
- 开轨 $\Rightarrow$ 无界/奇异/单调型（$\Gamma_7$）
    
### 6.9 汇总表

- $h<h_{\min}$：$\Gamma_6$：无实行波
    
- $h=h_{\min}$：$\Gamma_5$：$u\equiv 2c/b$
    
- $h_{\min}<h<0$：
     - $\Gamma_3$（周期波）：$\varphi(\xi)=\alpha-(\alpha-\beta) \mathrm{sn}^2\Big(\kappa\xi,m_1\Big)$
     - $\Gamma_4$：$\varphi(\xi)= \frac{\gamma-\beta\mathrm{sn}^2(\kappa\xi,m_1)}{1-\mathrm{sn}^2(\kappa\xi,m_1)}$
     
- $h=0$：
    - $\Gamma_1$（孤立波）：$u=\dfrac{3c}{b}\mathrm{sech}^2\big(\frac{\sqrt c}{2}(x-ct)\big)$
        
    - $\Gamma_2$：$u=-\dfrac{3c}{b}\mathrm{csch}^2\big(\frac{\sqrt c}{2}(x-ct)\big)$
        
- $h>0$：$\Gamma_7$：开轨（无界/奇异/单调型）

其中 $h_{\min}=-\dfrac{2c^3}{3b^2}$（当 $A=0$）。

  Block[{u, x, t, \[Phi], \[Xi], y, pde, ode3, ode2, sysEq, H, V,
